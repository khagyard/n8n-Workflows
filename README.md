Retrieves support tickets and generates draft support responses based on ticket details and a knowledge base

**Skills Demonstrated**

 - API integration
 - Structured AI outputs
 - Workflow Orchestration

### Weekly Support Ticket Summary
Aggregates supprt ticket data and generates and emails AI-powered summaries highlighting trends and ticket statistics.

**Skills Demonstrated**
- Data Aggregation
- Prompt engineering for summarization
- HTML Formatting
- API Integration
- Reporting

Automatically retrives support ticket data and searches the knowledge base for relevant articles to be returned in the ticket reponse.

**Skills Demonstrated**
- API Integration
- Prompt engineering
- AI assisted ranking
- Data transformation
- Javascript coding
- JSON Parsing
